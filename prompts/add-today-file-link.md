# 오늘 날짜 파일 링크 추가 프롬프트

## 개요
오늘 날짜 이름의 HTML 파일을 찾아서 index.html에 링크로 추가하는 프롬프트입니다.

## 프롬프트
> @index.html 문서에 오늘 날짜로 생성된 HTML 파일을 찾아 링크를 추가해줘. 파일명이 YYYY-MM-DD.html 형식이면 해당 파일을 기준으로 링크를 넣어줘.

## 처리 방식
- 현재 날짜를 기준으로 YYYY-MM-DD.html 패턴의 파일을 찾습니다.
- 해당 파일이 있으면 index.html에 새 링크를 추가합니다.
- 링크 텍스트는 예를 들어 "2026-07-09 실습"처럼 표시합니다.
- 이미 같은 링크가 있으면 중복 추가하지 않습니다.

## 적용 코드 예시
```html
<li><a href="2026-07-09.html">2026-07-09 실습</a></li>
```

## 참고 사항
- 파일명은 YYYY-MM-DD.html 형식으로 가정합니다.
- 루트 폴더 기준으로 링크 경로를 작성합니다.
- index.html의 기존 목록 섹션에 자연스럽게 추가하면 좋습니다.
- 필요하면 "오늘 학습" 같은 새 섹션을 만들어도 됩니다.
