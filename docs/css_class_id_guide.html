🎨 클래스(class) vs 아이디(id)

웹페이지 요소를 꾸미는 두 가지 비밀 열쇠!

🆔 아이디 (id) - "너는 특별해!"

👤

아이디는 여러분의 '이름'이나 '주민번호'와 같아요.
반에서 이름이 똑같은 사람이 있을 순 있지만, 한 명을 딱 짚어서 부를 때 쓰는 유일한 별명이죠!

  • ✨ 한 페이지에 딱 한 번만 쓸 수 있어요.
  • ✨ CSS에서는 기호 # (샵)을 붙여서 불러요.
  • ✨ 가장 힘이 세서 다른 스타일보다 먼저 적용돼요.
#my-name { color: red; }
<p id="my-name">안녕, 나는 준서야!</p>

👕 클래스 (class) - "우리 같이 입자!"

👯‍♂️

클래스는 학교의 '교복'이나 '운동복'과 같아요.
여러 명의 친구가 똑같은 옷을 입고 있으면 한꺼번에 "빨간 팀 모여라!" 하고 부를 수 있죠?

  • ✨ 여러 개의 요소에 중복해서 쓸 수 있어요.
  • ✨ CSS에서는 기호 . (점)을 붙여서 불러요.
  • ✨ 여러 가지 스타일(옷)을 겹쳐 입을 수도 있어요.
.red-team { color: red; }
<p class="red-team">철수 (빨간 팀)</p>
<p class="red-team">영희 (빨간 팀)</p>

🆚 한눈에 비교하기

특징 🆔 아이디 (id) 👕 클래스 (class)
비유 이름, 주민번호 교복, 팀 컬러
사용 횟수 딱 1번만! 여러 번 가능!
CSS 기호 # (샵) . (점)
힘(우선순위) 매우 강함 💪 보통 👍
💡 선생님의 꿀팁!
보통 웹페이지를 만들 때는 클래스(.class)를 많이 사용해요.
나중에 똑같은 스타일을 다른 곳에도 쓰고 싶을 때가 많거든요!
아이디는 정말 특별한 한 곳(예: 맨 위로 가기 버튼 등)에만 아껴서 사용하세요.
