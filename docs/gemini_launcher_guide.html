Gemini CLI 런처 사용 설명서

생성된 gemini-launcher.bat 파일을 사용하여 원하는 경로에서 Gemini CLI를 빠르고 간편하게 실행하는 방법을 안내합니다.

주요 실행 방법

1. 드래그 앤 드롭 (가장 간편한 방법)

Windows 탐색기에서 원하는 폴더를 마우스로 잡고 gemini-launcher.bat 파일 위로 끌어다 놓으세요.

  • 해당 폴더 경로에서 즉시 Gemini CLI가 실행됩니다.

2. 직접 경로 입력

gemini-launcher.bat 파일을 더블 클릭하여 실행합니다.

  1. 콘솔 창에 Path: 라는 문구가 뜹니다.
  2. 원하는 전체 경로(예: C:\Users\Documents\MyProject)를 복사해서 붙여넣고 Enter를 누르세요.

3. 현재 폴더에서 실행

gemini-launcher.bat 파일이 위치한 폴더에서 바로 작업을 시작하고 싶을 때 사용합니다.

  • 파일을 더블 클릭한 후, 경로를 입력하지 않고 바로 Enter를 누르세요.

활용 팁

💡 바탕화면 바로가기 만들기
gemini-launcher.bat 파일을 우클릭하여 [보내기] -> [바탕화면에 바로가기 만들기]를 선택하세요. 이제 폴더를 작업표시줄이나 바탕화면의 아이콘으로 끌어다 놓기만 하면 어디서든 Gemini를 소환할 수 있습니다!
⚠️ 참고 사항
입력한 경로에 한글이나 공백이 포함되어 있어도 큰따옴표 없이 입력하거나 붙여넣으시면 스크립트가 자동으로 처리합니다.
