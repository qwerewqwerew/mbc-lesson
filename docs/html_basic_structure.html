초등학생도 이해하는 HTML

웹페이지 만드는 법, 쉽게 알려줄게!

❓ HTML이 뭐예요?

HTML은 웹페이지를 만드는 언어야!

책을 생각해봐. 책에는 표지, 목차, 본문이 있지? HTML도 똑같아!

📖 책 = HTML 문서

책의 부분 HTML 부분 설명
표지 <!DOCTYPE html> "이 책은 HTML5 책이에요!" 라고 알려줘
책 전체 <html> 책의 첫 장부터 마지막 장까지
목차/저자 정보 <head> 책 안에 적혀 있지만, 본문은 아닌 부분
본문 내용 <body> 실제로 읽는 이야기 내용

🏠 집으로 비유하면 더 쉬워!

🏡 HTML = 집 한 채

📋
<!DOCTYPE html>
"이 집은 현대식 집이에요!" (건축 스타일 알려주기)
🏠
<html>
집 전체 (땅부터 지붕까지)
📦
<head>
지하 창고 (눈에 안 보이는 집 정보)
🚪
<title>
집 문 앞 이름표 ("준서네 집")
🛋️
<body>
실제로 사는 공간 (거실, 방, 주방)

✏️ 초등학생용 예시 코드

<!-- 이건 주석: 사람만 보고 컴퓨터는 무시해 --> <!DOCTYPE html> <html> <head> <title>나의 첫 웹페이지</title> </head> <body> <h1>안녕! 나는 준서야</h1> <p>나는 초등학교 4학년이야.</p> <p>좋아하는 것: 피자 🍕, 게임 🎮, 축구 ⚽</p> </body> </html>

🎯 외워야 할 것 (딱 4개!)

  1. <html> → 집 전체
  2. <head> → 눈에 안 보이는 정보 창고
  3. <title> → 문 앞 이름표
  4. <body> → 실제로 보이는 방

🔧 head에 추가할 수 있는 것들

태그 설명
<link> 외부 CSS 파일 연결 (집 꾸미기 도구)
<script> JavaScript 코드 삽입 (집에 기능 추가)
<style> 내부 CSS 스타일 정의 (벽 색, 글씨 크기)
<meta> SEO, 저자, 키워드 등 추가 정보

🚀 완전한 예시

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width"> <meta name="description" content="HTML 기본 구조 설명"> <title>HTML 기본 구조</title> <link rel="stylesheet" href="style.css"> </head> <body> <header> <h1>웹 페이지 제목</h1> </header> <main> <p>메인 콘텐츠 영역</p> </main> <footer> <p>© 2026</p> </footer> <script src="script.js"></script> </body> </html>
