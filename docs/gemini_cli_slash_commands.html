Gemini CLI Slash Commands Documentation

Gemini CLI에서 사용할 수 있는 주요 슬래시 명령어들을 카테고리별로 정리한 문서입니다.

Core Commands

/about
현재 CLI의 버전 정보를 표시합니다.
/auth
인증 방법을 변경할 수 있는 대화 상자를 엽니다.
/bug
Gemini CLI에 대한 GitHub 이슈를 생성합니다. 명령어 뒤에 오는 텍스트가 이슈 제목이 됩니다.
/clear
터미널 화면과 스크롤백을 지웁니다 (단축키: Ctrl+L).
/docs
웹 브라우저에서 Gemini CLI 공식 문서를 엽니다.
/help (또는 /?)
사용 가능한 명령어에 대한 일반적인 도움말 정보를 표시합니다.
/quit (또는 /exit)
CLI를 종료합니다. --delete 플래그를 사용하면 현재 세션 기록과 임시 파일을 영구적으로 삭제합니다.
/settings
Gemini CLI 설정을 수정할 수 있는 대화형 편집기를 엽니다.
/stats
상세 세션, 모델, 도구 사용 통계를 표시합니다.
/upgrade
브라우저에서 Gemini Code Assist 업그레이드 페이지를 엽니다.

Session & History Management

/resume (alias: /chat)
자동 저장된 세션을 탐색할 수 있는 브라우저를 엽니다. 수동 체크포인트(save, list, resume, delete, share)도 지원합니다.
/rewind
대화 내역을 뒤로 이동하여 상호작용을 검토하거나 코드 변경 사항을 되돌립니다 (단축키: Esc Esc).
/restore [id]
특정 도구 실행 전의 상태로 프로젝트 파일을 복구합니다.
/compress
토큰 절약을 위해 전체 채팅 컨텍스트를 요약으로 대체합니다.
/copy
Gemini의 마지막 출력을 시스템 클립보드에 복사합니다.

Context & Configuration

/directory (/dir)
멀티 디렉토리 지원을 위한 워크스페이스 디렉토리를 관리합니다 (add, show).
/init
현재 디렉토리를 분석하고 프로젝트 전용 지침을 위한 GEMINI.md 파일을 생성합니다.
/memory
GEMINI.md 파일의 계층적 지침 컨텍스트를 관리합니다 (add, list, refresh, show).
/model
활성 모델을 구성합니다 (manage, set <model-name>).
/plan
계획 모드(Plan Mode)로 전환하거나 현재 계획을 복사합니다.
/tools
사용 가능한 도구 목록을 표시합니다 (desc: 상세 설명, nodesc: 이름만).
